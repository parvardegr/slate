    Introduction

    Welcome to the GraphVisualization API! You can use our API to access GraphVisualization API endpoints.

    Reports

    Get List Reports

    This endpoint retrieves all reports in a same group.

    returns JSON structured like this:

    [{
    "name": "report name",
    "extuid": "uid_019",
    "nodes_count": 12,
    "edges_count": 11,
    "preview": "image_hash"
  },
  {
    "name": "report name",
    "extuid": "uid_120",
    "nodes_count": 5,
    "edges_count": 2,
    "preview": "image_hash"
  }
]

    HTTP Request

    GET http://example.com/api/report/items

    Query Parameters

    Parameter Required Default Description
    group Required group uid.
    start Optional 1 page number.
    len Optional -1 page size. set -1 to get all
    this method will give you light dto of reports.

    Get a Specific Report

    This endpoint retrieves a specific Report.

    returns JSON structured like this:

    {
  "name": "report name",
  "group": "uid_102",
  "creator": "username",
  "zoom": 50.0,
  "content": {
    "data": ...
  },
  "extuid": "uid_607",
  "owner": "uid_202",
  "nodes_count": 23,
  "edges_count": 331266643,
  "create_date": 1511349154435,
  "last_modifie": 1511349154435,
  "preview": "IMAGE_HASH"
}

    HTTP Request

    GET http://example.com/api/report/items/{uid}

    URL Parameters

    Parameter Required Default Description
    uid Required report uid.
    this method will give you full dto of report.

    Delete a Specific Report

    This endpoint deletes a specific Report.

    HTTP Request

    DELETE http://example.com/api/report/items/{uid}

    URL Parameters

    Parameter Required Default Description
    uid Required report uid.
    this method return void.

    Edite a Specific Report

    This endpoint edite a specific Report.

    HTTP Request

    POST http://example.com/api/report/items/{uid}

    BODY

    json that contain changes

    POST Body JSON structured like this:

    {
  "name": "changed_name",
  "group": "chandeg_group_uid",
}

    URL Parameters

    Parameter Required Default Description
    uid Required report uid.
    this method return void.

    Add Report

    This endpoint add a new report.

    returns JSON structured like this:

    {
    "name": "report name",
    "extuid": "uid_019",
    "nodes_count": 12,
    "edges_count": 11,
    "preview": "image_hash"
}

    HTTP Request

    POST http://example.com/api/report/items

    Query Parameters

    Parameter Required Default Description
    group Required group uid.
    name Required report name.
    this method will add new report with current date.

    Search

    This endpoint search in reports(on name field only).

    returns JSON structured like this:

    [{
    "name": "matched_name",
    "extuid": "uid_019",
    "nodes_count": 12,
    "edges_count": 11,
    "preview": "image_hash"
  },
  {
    "name": "matched_name",
    "extuid": "uid_120",
    "nodes_count": 5,
    "edges_count": 2,
    "preview": "image_hash"
  }
]

    HTTP Request

    GET http://example.com/api/report/search

    Query Parameters

    Parameter Required Default Description
    query Required a simple string to search in name field.
    start Optional 1 page number.
    len Optional -1 page size. set -1 to get all
    this method will give you light dto of reports.

    HTTP Request

    URL Parameters

    Parameter Required Default Description

    Errors

    The Graph API uses the following error codes:

    Error Code Meaning
    400 Bad Request -- Your request sucks.
    401 Unauthorized -- Your API key is wrong.
    403 Forbidden -- The kitten requested is hidden for administrators only.
    404 Not Found -- The specified kitten could not be found.
    405 Method Not Allowed -- You tried to access a kitten with an invalid method.
    406 Not Acceptable -- You requested a format that isn't json.
    410 Gone -- The kitten requested has been removed from our servers.
    418 I'm a teapot.
    429 Too Many Requests -- You're requesting too many kittens! Slow down!
    500 Internal Server Error -- We had a problem with our server. Try again later.
    503 Service Unavailable -- We're temporarily offline for maintenance. Please try again later.
